**Customer Churn Data Visualization**
Project Overview
This project analyzes a telecommunications customer churn dataset using Microsoft Excel. It focuses on visualizing customer behavior and identifying churn patterns through charts and PivotTables.

**Tools Used**
Microsoft Excel
PivotTables
Column Charts
Histogram
Clustered Bar Chart

**Tasks Completed**
Data Overview and Sorting
Churn Count Visualization
Monthly Charges Distribution
Churn Rates by Gender

**Skills Gained**
Data Cleaning
Data Visualization
PivotTables
Excel Chart Creation
Data Analysis

**Outcome**
Created interactive visualizations to understand customer churn trends and support business decision-making.
